# Typing Test

A modern typing speed test application built with React, TypeScript, and Vite.

## Features

- Real-time WPM and accuracy tracking
- Multiple duration options (15s, 30s, 60s, 120s)
- Clean, minimal UI inspired by Monkeytype
- Character-by-character feedback

## Tech Stack

- **Frontend:** React 18 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Package Manager:** pnpm

## Getting Started

### Install dependencies
```bash
pnpm install
```

### Run development server
```bash
pnpm dev
```

### Commit changes
```bash
pnpm commit
```
